An API dedicated to kitty cats <3

Kit-API is a project I started when another cat fact api that I used for a while went down for a few days. Hopefully this project can provide cat facts to someone else in their time of need. The API is completely free to use.

Github 
import aiohttp
import asyncio


async def main() -> None:
    URL = "https://kit-api.com/v1/fact"

    async with aiohttp.ClientSession() as session:
        async with session.get(URL) as response:
            if response.ok:
                print(await response.json())
            else:
                print(f"Error: {response.status}")


if __name__ == "__main__":
    asyncio.run(main())
